# 🏠 EstateAI

## AI-Powered Real Estate Valuation Platform

EstateAI is a full-stack AI-based real estate valuation platform that predicts property prices using machine learning and generates professional valuation reports.

## 🚀 Features

✨ AI Property Price Prediction  
📊 Interactive Dashboard  
📄 PDF Valuation Report Generation  
🔐 User Authentication & Profile Management  
⭐ Community Reviews & Feedback System  
📈 Prediction History Tracking  
🌙 Modern Dark AI SaaS UI  
📱 Responsive Design  

## 🤖 Machine Learning

EstateAI uses machine learning techniques to estimate property values based on:

- Property area
- Location
- Bedrooms
- Bathrooms
- Year built
- Garage availability
- Other property features

Model:
- Random Forest Regression
